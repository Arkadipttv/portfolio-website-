WiseShield AI

Hack4Bengal 3.0 Project

WiseShield AI is a robust phishing detection platform combining Machine Learning, a Flask API server, and a Chrome Extension. It detects login fields, sends data to the server, and validates website authenticity in real-time. If found malicious, it alerts and redirects the user. Beware of phishing.

EcoGrow

FrostHacks 1.0 Project

EcoGrow enables users to support environmental sustainability by earning tokens through ad views. These tokens sponsor real-world tree planting. Compete on the leaderboard and earn fresh produce as monthly rewards while helping NGOs make a greener planet.
